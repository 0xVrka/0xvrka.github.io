# 0xvrka's Blog

My personal blog powered by [Astro](https://astro.build) and the [Yukina](https://github.com/WhitePaper233/yukina) theme.

[**Visit Live Site**](https://0xvrka.github.io)

## Run Locally

Requires **Node.js v22+** and **pnpm**.

```bash
# Install
corepack enable
pnpm install

# Dev
pnpm dev

# Build
pnpm build

```
